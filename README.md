# Book CSV Manager

React + TypeScript app for uploading, editing, filtering, sorting, and downloading large CSVs (~10k rows) of books. Uses PapaParse, TailwindCSS, and react-window.

## Scripts

- `npm run dev` – start dev server
- `npm run build` – production build
- `npm run preview` – preview build

## Setup

```bash
npm i
npm run dev
```

If the CSV is not uploaded, use "Generate 10,000 Fake Rows" to test.

## CSV Columns

- Title
- Author
- Genre
- PublishedYear
- ISBN

## Features

- CSV upload via PapaParse
- Virtualized table with editable cells
- Change highlighting per cell and per row
- Global search across columns
- Column sorting
- Pagination and row count
- Download edited CSV
- Reset edits to original
- Responsive Tailwind UI
